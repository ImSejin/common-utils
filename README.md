# Getting started

### Maven

```xml
<dependency>
  <groupId>io.github.imsejin</groupId>
  <artifactId>common-utils</artifactId>
  <version>${common-utils.version}</version>
</dependency>
```

### Gradle

```groovy
implementation group: "io.github.imsejin", name: "common-utils", version: "$commonUtilsVersion"
```
